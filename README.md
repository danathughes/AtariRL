# AtariRL

Implementations of various Deep-Q Networks for learning Atari games.

## Prerequisites

The software and versions used to implement the code are

1.  Python 2.7

2.  Tensorflow 1.2

3.  Numpy 1.13

4.  pyGame 1.9.1
